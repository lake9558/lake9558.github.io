Ubuntu-linux挂载、卸载及格式化U盘

1 找出U盘所在分区

不插U盘执行
sudo fdisk -l 输入密码回车
插上U盘执行
sudo fdisk -l
多出来的那一块就是U盘所在分区
这里以我U盘所在分区 /dev/sdb4为例
2 挂载U盘
sudo mount /dev/sdb4 /mnt
(这里表示将U盘挂载在mnt目录下）

3 卸载U盘
sudo umount /dev/sdb4

4 将U盘格式化为fat32格式

sudo mkfs.vfat /dev/sdb4

将U盘格式化为ntfs格式

sudo mkfs.ntfs /dev/sdb4
