ubuntu下添加交换空间

wap查看交换空间情况
mkdir swap
cd swap
sudo dd if=/dev/zero of=swapfile bs=1024 count=4000000
count为总量，这里正好为4g
启用交换空间sudo swapon swapfile
设置权限 sudo chmod 600 swapfile
设置自启动 sudo gedit /etc/fstab
添加一行: /home/lake/swap swap defaults 0 0
最后free -m 查看交换空间情况
